Privacy Policy

This application does not collect, store, or share personal information of users. However, in-game data and scores may be stored on Firebase Realtime Database.

Collected Data

The app only collects anonymous usage data and in-game scores to improve gameplay experience, which are stored securely on Firebase Database.

Third-Party Services

The app uses third-party services such as Google Play Services and Firebase for analytics and cloud storage.

Data Security

User data is used solely to enhance gameplay and is not shared with third parties. Firebase security rules protect all stored data.

Target Audience

This game is intended for users aged 6 and above. Parental guidance is recommended for children under 13.

Safety Notice

This is a parkour-style game. Users should not attempt to replicate any stunts or movements shown in the game in real life. Play safely.
